# Meesho Admin Panel

Runs on **port 5173**

## Setup
```bash
npm install
npm run dev
```

## Auth Flow
```
/login  →  (admin credentials)  →  /dashboard
```

## Routes
| Path | Page |
|------|------|
| /login | Admin Login |
| /dashboard | Dashboard |
| /products | Products |
| /orders | Orders |
| /categories | Categories |
| /users | Users |
| /payments | Payments |
| /change-password | Change Password |
